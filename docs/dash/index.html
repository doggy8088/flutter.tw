這是 Dash：


Dash 是 Dart 語言和 Flutter 框架的吉祥物。

info 提示：

你可以透過 Dashatar 這個網頁端和行動端的 Flutter 應用定製 你自己的 Dash 虛擬形象。


Dash 的由來？

Shams Zakhour 從 2013 年 12 月在 Google 擔任 Dart 作者的職務開始，她就一直在倡導推進 Dart 的吉祥物。在為 Java 撰寫了 14 年的文件後，她發現 Java 的吉祥物 Duke 十分受歡迎，從而希望讓 Dart 也擁有類似的東西。

但這個想法直到 2017 年一位 Flutter 工程師 Nina Chen 在內部郵件中再次提出，才取得了實質性的進展。時任副總裁 Joshy Joseph 批准了這個想法，並且讓 2018 年度 Dart 會議 的組織者 Linda Rasmussen 進行落地。

Shams 在知曉這個計劃後，立刻找到了 Linda，希望主導並推進這個專案，計劃生產一些毛絨玩具。 Linda 早前已經設計出了一些草圖，她將這些草圖移交給了 Shams。有了這些草圖後，Shams 找到了一家可以在截止日期前（趕在農曆新年之前）合作的供應商，接著便開始製作毛絨玩具的規格。

是的，Dash 最初是 Dart 的吉祥物，而不是 Flutter 的。

為什麼是一隻蜂鳥？

最初，蜂鳥用來作為展示和網頁用途。使用蜂鳥代表 Dart 是一門速度極快的語言。


然而，又尖又有稜有角的蜂鳥無法作為毛茸茸的可愛玩具，所以我們最終選擇了圓潤的蜂鳥。

Shams 為 Dash 指定了顏色的應用範圍、尾巴的形狀、一小簇頭髮、眼睛以及其他小細節。供應商將這些細則發給兩個製造商後，他們在幾周後送回了毛絨玩具的原型。


在 2018 Dart 會議上介紹 Dash。

在製造商推進製作的同時，Shams 毛絨玩具取了個名字：Dash，它其實是 Dart 專案的早期代號，而且不分性別，似乎更適合作為一隻蜂鳥的名字。

許多箱 Dash 毛絨玩具趕在會議開始前送到了南加州。它們受到了大量 Dart 和 Flutter 愛好者的喜愛。

Flutter 和 Dart 的演講者都攜帶著 Dash 進行了演講，所以 Dash 就成為了 Flutter 和 Dart 的吉祥物。

會議上的 Dash

自 Dash 1.0 之後，我們又製作了兩個版本的 Dash。市場部門針對 Dash 1.0 微調了 Dart 和 Flutter 的配色方案， Dash 2.0 對應了更新後的配色方案（移除了綠色）。 Dash 2.1 尺寸更小，並且調整了更多配色。小尺寸的 Dash 更易運輸，並且更適合放在娃娃機中！
